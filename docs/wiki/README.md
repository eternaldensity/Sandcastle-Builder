## Cross-Linking Convention

This documentation uses Obsidian-style wikilinks:

- `[[Boost Name]]` - Links to boost page
- `[[Badge Name]]` - Links to badge page
- `[[glossary#term]]` - Links to glossary term
- `[[mechanics/glass]]` - Links to mechanics page
