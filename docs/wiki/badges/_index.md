# Badges

> Source: [Badges](https://xkcd-time.fandom.com/wiki/Badges)

Badges are achievements, used both as a guide on what to do within the game [[Sandcastle_Builder|Sandcastle Builder]], and as multipliers on sand collection via various [[Boosts|Boosts]]. There are currently 229 Badges, and 1010 [[Discoveries|Discoveries]] (which each can count as six badges each using the proper tools).
