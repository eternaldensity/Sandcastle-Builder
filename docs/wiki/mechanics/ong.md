# ONG

> Source: [ONG](https://xkcd-time.fandom.com/wiki/ONG)

The **ONG** (also GONG, BONG, DONG, or PONG) (but actually DING) is a timer established to measure time between updates of the [[One_True_Comic|One True Comic]]. The gong is set to alert followers of an immenent change in [[Frame|frames]]. Users who react quick enough may see the change happen before their eyes. Only a single user may be able to post the new frame first, however the chance for becoming a Ninja is great.

The GONG site with most of a Timeframe left until the newpic.

The delay on the gong is approx. 0.053 [[TimeFrames|standard grains]] before the actual change in image. The gong provides the basis for the [[TimeFrames|TimeFrames]] standard unit of measure.

The website for the gong is [http://www.webcountdown.de/?a=gxMXfoy](http://www.webcountdown.de/?a=gxMXfoy)

The website for the [[Longpix|longpix]] version of the gong is [http://www.webcountdown.net/?a=v2WZqAO&k=H68Q](http://www.webcountdown.net/?a=v2WZqAO&k=H68Q)
